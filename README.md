👨‍💻 Omiros Loupis - Personal Portfolio

A modern, fully responsive personal portfolio website designed to showcase my software engineering projects and technical skills.
Built with a mobile-first approach using HTML5, Tailwind CSS, and Vanilla JavaScript.

🔗 Live Demo: https://omiros-loupis.github.io/Portfolio/

✨ Key Features

🎨 Modern UI/UX: Utilizes "Glassmorphism" design, gradients, and a clean typography system (Inter font).

🌓 Dark Mode: Fully functional theme toggle that persists user preference via the browser's LocalStorage API.

⚡ Performance: Optimized for speed with zero external heavy frameworks (No React/Angular overhead for a static site).

📱 Responsive Design: Custom mobile navigation and fluid layouts ensuring perfect rendering on all devices.

✨ Animations: Smooth scroll reveals implemented using the Intersection Observer API (no jQuery).

🛠️ Tech Stack

Frontend: HTML5, CSS3

Styling: Tailwind CSS (CDN)

Scripting: Vanilla JavaScript (ES6+)

Icons: Lucide Icons

Deployment: GitHub Pages
